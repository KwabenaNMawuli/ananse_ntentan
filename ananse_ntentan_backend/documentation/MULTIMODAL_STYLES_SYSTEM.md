## üé¨ Video Generation Styles (Future Feature!) ‚≠ê

### Purpose:
If/when adding video output generation:
- **Animation style** - How panels are animated
- **Transition effects** - How panels flow together
- **Pacing** - Speed and rhythm of the video

### Video Style Categories:

#### 1. **Motion Comic**
- **Animation**: Subtle panel movements, Ken Burns effect
- **Transitions**: Smooth fades, cinematic wipes
- **Pacing**: Slow, cinematic (30-60s per panel)
- **Effects**: Parallax depth, light effects
- **Best For**: Dramatic stories, cinematic feel
- **Reference**: Motion comics, digital comics

#### 2. **Animated Storyboard**
- **Animation**: Limited animation, key frame transitions
- **Transitions**: Quick cuts, storyboard-style
- **Pacing**: Fast, energetic (10-15s per panel)
- **Effects**: Speed lines, action bursts
- **Best For**: Action, fast-paced stories
- **Reference**: Animatics, storyboards

#### 3. **Slideshow Presentation**
- **Animation**: Static panels, clean transitions
- **Transitions**: Simple fades, dissolves
- **Pacing**: Moderate (20-30s per panel)
- **Effects**: Minimal, clean, professional
- **Best For**: Educational, straightforward narratives
- **Reference**: PowerPoint, Keynote presentations

#### 4. **Anime Opening Style**
- **Animation**: Dynamic zooms, dramatic poses
- **Transitions**: Fast cuts, dramatic reveals
- **Pacing**: Fast, music-driven (5-10s per panel)
- **Effects**: Flash frames, dramatic lighting
- **Best For**: Action, adventure, dramatic stories
- **Reference**: Anime openings, AMVs

#### 5. **Documentary Style**
- **Animation**: Pan and scan, archival footage feel
- **Transitions**: Subtle fades, professional
- **Pacing**: Informative, measured (30-45s per panel)
- **Effects**: Film grain, vintage look (optional)
- **Best For**: True stories, historical narratives
- **Reference**: Ken Burns documentaries

#### 6. **TikTok/Reels Fast-Cut**
- **Animation**: Quick zooms, fast pans
- **Transitions**: Jump cuts, snap transitions
- **Pacing**: Very fast (3-5s per panel)
- **Effects**: Modern, trendy, text overlays
- **Best For**: Short-form content, viral stories
- **Reference**: TikTok, Instagram Reels

#### 7. **3D Parallax**
- **Animation**: 3D depth effect, layer separation
- **Transitions**: Smooth parallax scrolling
- **Pacing**: Smooth, immersive (20-30s per panel)
- **Effects**: Depth blur, 3D movement
- **Best For**: Immersive stories, modern feel
- **Reference**: Parallax websites, 3D comics

#### 8. **Glitch Art Experimental**
- **Animation**: Glitchy, distorted, abstract
- **Transitions**: Glitch effects, digital artifacts
- **Pacing**: Variable, unpredictable
- **Effects**: Datamosh, chromatic aberration, digital noise
- **Best For**: Experimental, cyberpunk, abstract
- **Reference**: Glitch art, vaporwave aesthetics

### Audio Generation (NEW): ‚≠ê

#### Option 1: Google Cloud TTS (Recommended)
```javascript
const textToSpeech = require('@google-cloud/text-to-speech');

const client = new textToSpeech.TextToSpeechClient();

const request = {
  input: { text: story.aiNarrative },
  voice: {
    languageCode: 'en-US',
    name: 'en-US-Neural2-J',  // Deep male voice
    ssmlGender: 'MALE'
  },
  audioConfig: {
    audioEncoding: 'MP3',
    speakingRate: 0.95,  // Slightly slower for drama
    pitch: -2.0,         // Deeper voice
    effectsProfileId: ['headphone-class-device']
  }
};

const [response] = await client.synthesizeSpeech(request);
```

**Pros**: Free tier, high quality, many voices  
**Cost**: $4 per 1M characters (very cheap)

#### Option 2: ElevenLabs (Premium Quality)
```javascript
const response = await fetch('https://api.elevenlabs.io/v1/text-to-speech/{voice_id}', {
  method: 'POST',
  headers: {
    'xi-api-key': process.env.ELEVENLABS_API_KEY,
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    text: story.aiNarrative,
    model_id: 'eleven_multilingual_v2',
    voice_settings: {
      stability: 0.5,
      similarity_boost: 0.75,
      style: 0.5,          // Dramatic style
      use_speaker_boost: true
    }
  })
});
```

**Pros**: Hollywood-quality voices, emotional  
**Cons**: More expensive (~$0.30/1K characters)

#### Option 3: OpenAI TTS (Good Balance)
```javascript
const mp3 = await openai.audio.speech.create({
  model: "tts-1-hd",
  voice: "onyx",  // Deep, authoritative
  input: story.aiNarrative,
  speed: 0.9      // Slightly slower
});
```

**Pros**: Good quality, simple API  
**Cost**: $15 per 1M characters
